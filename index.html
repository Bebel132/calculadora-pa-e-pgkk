Fórmula do Termo Geral da PA:

an = a1 + (n-1) * r

Onde:

an = Enésimo termo/último termo
a1 = Primeiro termo
n = Número de termos
r = Razão

Fórmula da Soma dos Termos de uma PA:
Sn = ((a1 + an) * n) / 2

Onde:

a1 = Primeiro termo
an = Enésimo termo/último termo
n = Número de termos

Fórmula do Termo Geral de uma PG:
an = a1 * a^n-1

Onde:

an = Enésimo termo/último termo
a1 = Primero termo
q = Razão
n = Quantidade de termos

Fórmula da soma do n termos de uma PG
Sn = (a1 * (q^n - 1)) / q - 1
